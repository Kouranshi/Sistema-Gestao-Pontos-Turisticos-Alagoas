# Sistema-Gestao-Pontos-Turisticos-Alagoas
Sistema acadêmico de gestão de pontos turísticos em Alagoas, desenvolvido em Python e PostgreSQL. Permite cadastrar, listar e avaliar locais turísticos, demonstrando integração entre banco de dados relacional e lógica de programação para fins de estudo e portfólio.


É necessário criar uma .env para haver a conexão com o banco de dados. Nesta .env deve conter as seguintes informações:

DB_HOST = SEU_HOST
DB_NAME = sistema_turismo_alagoas
DB_USER = SEU_USUARIO
DB_PASSWORD = SUA_SENHA
DB_PORT = SUA_PORTA
